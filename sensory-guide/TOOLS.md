# TOOLS.md — Sensory Guide 🎨

## Sensory Profile Questionnaire (Guided)
Ask parents to describe each system:
- What does the child avoid or react strongly to?
- What does the child seek out, crave, or do repeatedly?
- What sensory input seems to calm them?
- What sensory input sets them off?
- Any patterns — time of day, regulation state, environment?

## Sensory Modulation Scale (Simple)
Red = overwhelmed / shutting down
Yellow = dysregulated / seeking / agitated  
Green = regulated / available for learning/connection
Blue = under-aroused / zoned out

Tools to move UP from blue: vestibular, proprioceptive input, cold water, bright light
Tools to move DOWN from red/yellow: deep pressure, slow rhythm, dim light, quiet, darkness

## Recommended Tools & Products (I Can Describe, Not Endorse Brands)
- Noise-cancelling headphones (kids versions)
- Weighted blankets (1-3kg for children, consult OT for specifics)
- Fidget tools (various — match to sensory need)
- Chewelry (food-grade silicone chew necklaces)
- Compression clothing (base layers, seamless options)
- Liquid timer / sensory bottles (visual calming tools)
- Mini trampoline / wobble cushion / resistance bands

## Interoception Resources
- "The Interoception Curriculum" — Kelly Mahler
- Body-check activities: "How does my tummy feel? Is it empty or full?"
- Breathing awareness exercises (simplified)
- Temperature check-ins

## Fine Motor Support
- Handwriting Without Tears program (OT-developed)
- Pencil grip varieties: triangular, cushioned, adaptive
- Paper positioned at slight angle for writing comfort
- Typing as legitimate alternative for children with significant handwriting difficulties

## Collaboration Points
- **Emotional Coach:** Sensory overload triggers emotional dysregulation — always check sensory first
- **Routine Architect:** Sensory input needs to be built into routines proactively
- **Behaviour Advisor:** Many "behaviours" are sensory responses — understand before intervening
- **Communication Specialist:** Interoception difficulties affect emotional vocabulary development
