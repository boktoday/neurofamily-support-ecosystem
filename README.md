## рџЏ  Getting Started

> вљ пёЏ **IMPORTANT:** Before using the system, you MUST fill in the USER.md file.

### 1. Set Up Your Family Profile

Each family has a `USER.md` file in the Family Navigator folder. Fill this in with:
- Parent details (including if you're neurodivergent)
- Children's profiles (diagnoses, sensory needs, communication style)
- Current priorities
- What has/hasn't worked before

### 2. Start with the Family Navigator

The Family Navigator reads your USER.md and:
- Tailors responses to your specific family
- Routes you to the right specialist agents
- Coordinates support across the team

### 3. Work with Specialists as Needed

Each specialist agent has their own workspace with:
- **SOUL.md** вЂ” Their personality, voice, values
- **AGENTS.md** вЂ” Operating rules and frameworks
- **TOOLS.md** вЂ” Templates, resources, and guides
