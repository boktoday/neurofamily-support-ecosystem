## How to Use

1. **Set your region** in the Family Navigator's USER.md
2. Agents will load relevant country-specific resources
3. Crisis numbers, legal rights, and funding will be tailored to your location

---

## Contributing a Region

To add a new region:

1. Create a folder: `regional/[CODE]/`
2. Copy the template from `regional/_TEMPLATE.md`
3. Fill in country-specific details:
   - Diagnosis pathways
   - Funding systems
   - Education systems
   - Legal rights
   - Crisis contacts
   - Useful organizations

4. Update this index file
